# hero lab eeg research archive (biometric id + mtbi)

Archival research code from 2022, shared for reproducibility and transparency.
This repo includes EEG biometric identification work and related mTBI workflows from the same period.

Main files:
- `code/`
- `validation_loss_paradox.pdf`
- `how_i_detect_and_remove_eeg_artifacts.pdf`

Runs require local EEG datasets in the paths expected by the scripts.
